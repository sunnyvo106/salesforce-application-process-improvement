| Stakeholder Role                          | Role Description                                                                                                                                             | Reason for Involvement                                                                                                                               |
| ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| Business Analyst                          | Responsible for analysing the end-to-end application process, identifying pain points, eliciting requirements, and documenting findings and recommendations. | Oversees the project delivery, ensures alignment between business needs and technical solutions, and facilitates communication between stakeholders. |
| Salesforce Administrator                  | Manages and maintains the Salesforce platform, including system configuration, custom fields, automations, and data integrity.                               | Provides technical expertise, validates feasibility of solutions, and ensures changes align with system constraints and best practices.              |
| Recruitment Team (Subject Matter Experts) | End users responsible for reviewing applications, communicating with applicants, and managing recruitment workflows.                                         | Provide functional requirements, explain real-world processes, identify inefficiencies, and validate proposed solutions.                             |
| Applicant                                 | External user submitting an application through the system.                                                                                                  | Directly or indirectly impacted by changes to communication, workflow, and data handling within the application process.                             |
| Business Sponsor / Leadership             | Provides strategic direction, approves priorities, and makes final decisions on system changes.                                                              | Ensures solutions align with organisational goals, manages risk, and approves implementation of recommendations.                                     |
| Mentor / Advisor                          | Provides guidance, feedback, and professional support throughout the project lifecycle.                                                                      | Supports problem-solving, helps overcome blockers, and ensures learning outcomes and professional standards are met.                                 |
